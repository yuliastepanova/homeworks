2. Создать свой первый скрипт;
3. Закрепить пройденный материал:
- презентация и лекция;
- http://php.net/manual/ru/language.types.string.php#language.types.string.parsing
- Арифметические операторы http://php.net/manual/ru/language.operators.arithmetic.php

4. Написать PHP-скрипт, в котором создать текстовый массив с названиями цветов. 
Количество разных цветов должно быть больше четырех . Затем с помощью
PHP отобразить на странице четыре div одинакового размера. Сделать так, чтобы при загрузке страницы
все четыре div заливались случайными цветами, выбранными из массива. При этом, все четыре 
div должны заливаться разными цветами. Ни один цвет не должен повторяться. 
(Подсказака: функции массива shuffle, array_slice).
